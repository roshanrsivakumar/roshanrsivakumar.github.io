# The Life of a Showman — Roshan Sivakumar

Personal portfolio website for Roshan Rajkumar Sivakumar.

**Brand:** The Industrial Showman
**Stack:** Next.js 14 · Tailwind CSS · Framer Motion
**Live:** https://roshanrsivakumar.com

## Quick Start

```bash
npm install
npm run dev
```

## Deploy

Pushes to `main` automatically trigger a GitHub Actions build and deploy to `gh-pages` branch.

> **One-time setup:** Go to GitHub → Settings → Pages → Source → Deploy from a branch → `gh-pages` / `/ (root)`.

## Structure

